Frequently asked questions

My friend has created a game, how can I join it?

He should send you a room key, and you should paste it to the corresponding field

What is the meaning of the words, which are used as the room keys?

These "words" are just random sequences of letters, they don't have any meaning. No, you won't find any of them in a hat.

I've joined a room and I see a "Game not started" label. How can I start the game?

You are not the host of the game room, so you can't start the game. The host of the room is the one who created it. Only the host can do it.

How to enable voice chat?

It is not available yet. We assume that you are using some third party service for audio/video communication, such as Zoom, Skype or Discord.

Why is the round length equal to 40 seconds? Is it possible to adjust this value?

During testing, we found that the efficiency of the online explanations is worse, because it's difficult to speak and listen simultaneously. So, we've increased the round duration. It's possible to change it in the game settings. Only the host can do it.

What is the difference between the "Not explained" and "Mistake" buttons?

"Mistake" means that the speaker has violated the rules (e. g. said a cognate word). Pressing this button finishes the round and removes the word from the hat. "Not explained" button means that the speaker just didn't manage to explain the word, and the word stays in the hat.

How to change the number of words in the hat?

The host can do it, using game settings.

What is the logic behind the pair of speaker/listener choice?

The players can be arranged in a circle, using their joining order. Then, each player will be explaining words to the next one. After it, everyone will be explaining words to the next of the next player, and so on. Thus, after a while each pair of players will explain words to each other. When it happens, the process will be repeated.

Is it possible to explain words in predefined pairs?

Not yet, but thiss feature will be implemented soon.

When the game ends?

When all words are explained.

Is it possible to play some particular number of tours?

Not yet, but it will be implemented.

I've accidentally closed the game tab. What should I do?

You should enter the game room by its key with your name.

What is the difference between this hat and other similar apps?

Unlike other apps, you can play online with this hat, staying at home.

I don't like the sound. How can I turn it off?

It can be done in your personal settings.
